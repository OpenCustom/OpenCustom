# OpenCustom

OpenCustom is a modern, sleek landing page showcasing open-source development philosophy with interactive code visualization. The project demonstrates how open-source solutions can be tailored to specific business needs through expert adaptation and integration.

## üíª Technical Details

### Technologies Used
- HTML5 with semantic markup
- CSS3 with modern styling, CSS variables, and animations
- JavaScript (ES6+) for interactive features
- Fetch API for dynamic content loading
- CSS Grid and Flexbox for responsive layouts

### Code Animation System
The landing page features an advanced code animation system that:

- Loads code snippets from JSON configuration
- Provides real-time syntax highlighting
- Simulates realistic typing and erasing effects
- Cycles through multiple code examples automatically
- Tracks cursor position for immersive experience
- Supports various programming patterns and structures

### Supported Code Syntax
- TypeScript/React components
- Custom hooks and utilities
- API routes and handlers
- Type definitions and interfaces
- React Query integration examples
- Utility functions and helpers

## üîß Installation & Usage

### Quick Start
1. Clone the repository
2. Navigate to the project directory
3. Open `index.html` in your browser

### Development
To modify the project:

1. Edit `assets/json/code.json` (!IMPORTANT) to add or change code examples
2. Update syntax highlighting rules in `core.js`
3. Adjust animation speeds as needed
4. Modify CSS variables in `core.css` for styling changes
